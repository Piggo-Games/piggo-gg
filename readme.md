# Development

|package|description|
|--|--|
|`core`| engine & games
|`web`| webapp
|`server`| game server
|`docs`| js bundle & resources (served statically by GitHub Pages)

### setup

```bash
# install dependencies
bun install

# launch server and web
bun dev
```
